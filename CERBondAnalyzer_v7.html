Total Return Projector — 30d / 60d / 90d
Para cada instrumento: carry accrued + rolldown gain + CER appreciation (usando la grilla de inflación), neto de spread cost. Se rankea por total return proyectado. Responde: ¿qué conviene holdear?
Z-Spread a Curva NSS (Cashflow-based)
Z-Spread real: spread constante z tal que Σ CF_i / (1 + r_NSS(t_i) + z)^t_i = Precio mercado. Se resuelve por bisección sobre los cashflows reales del bono, descontados con la curva NSS fitted + z. Más riguroso que ΔTNA simple para bonos con diferentes estructuras.

Implied Forward Inflation Term Structure
Inflación mensual implícita por el mercado para cada mes futuro, derivada de breakeven CER vs LECAP comparable. Barras violeta = forward-forward entre bonos adyacentes (inflación implícita entre nodos). X = mes forward, Y = TEM implícita. Compara con tu proyección (grilla de inflación).
Forward Inflation Matrix — Shock por Mes
Elegís un mes a shockear y una grilla de inflaciones posibles. La matriz te muestra cuánto debería dar cada mes siguiente para que el breakeven acumulado hacia cada horizonte siga siendo el implícito del mercado. Ejemplo: si Marzo da 3.0%, cuánto debería dar Abril para sostener el BE de Mayo.
Sintético FX — Arbitraje ROFEX vs Tasa Pesos
Compara la tasa implícita devaluatoria del futuro ROFEX contra la tasa en pesos disponible. Si spread > 0 → conviene sintético largo (vender futuro + colocar pesos). Si spread < 0 → sintético corto (comprar futuro + tomar pesos).

Metodología

API BCRA v4: se mantiene exactamente el fetch del CER desde /estadisticas/v4.0/Monetarias/30. Este archivo sigue siendo HTML+JS ejecutable desde VS Code / navegador.

Ventanas CER: inflación de mes M alimenta CER del 16/M+1 al 15/M+2. Se interpola diario con ancla en día 15.

CER bono: referencia a 10 días hábiles antes de settlement y vencimiento, con feriados BYMA.

LECAP/BONCAP: se agregó un bloque nuevo, separado, sin romper la lógica CER original.

Tasa fija: VF = 100 × (1+TEM)meses 30/360, TEA = (VF/Precio)^(365/días) − 1, TNA = ((1+TEA)^(días/365) − 1) × 365/días.

Duration del scatter: modified duration aproximada para instrumentos bullet/capitalizables: Duration ≈ (días/365) / (1 + TEA). Para la curva CER se usa la TIR nominal equivalente; para tasa fija, la TEA calculada del título.

Regresión: ajuste logarítmico por mínimos cuadrados: y = a·ln(x) + b, con X = duration y Y = TNA.

Inflación implícita: se resuelve qué TEM promedio geométrica hace que el CER proyectado iguale al rendimiento del comparable en tasa fija. SOLO se incluyen bonos cuyos meses de inflación relevantes son aún desconocidos. Si el CER de referencia del bono cae antes del 15 del mes (ancla anterior = todo inflación conocida), el breakeven no tiene sentido y el bono se marca como "CER determinado".
